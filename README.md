# React portfolio

## Description

Portfolio website built with React and Redux

## Usage / Install

Clone repo, cd into folder and run:

```console
$ npm install
$ npm start
```

## Requirement

- npm

## Tools

- Create-react-app cli
- Redux
- Reselect
- React MDL material design
- Semantic UI React
- React Router v5
- React-Helmet
- Ress.css (Customized in html tag)
